Module 1:
Grammar
Here we’ll revise the new grammatical structures you’ve learned so far and study some grammar for articles!
You’ve learned Unreal conditionals and structures for Wishes and regrets. Let’s revise these grammar structures you can use in your articles!
    Present&future conditionals (imagined situation + predicted result):
  1. if-clause: could/Past Simple/Past Continious
  2. main clause: would/could/might + infinitive
If I were writing a final qualifying work, I would create an online-module for studying. If I could use LMS Moodle for it, I wouldn’t spend the New Year holidays creating a website.
    Past conditionals (possible situation + predicted result, they didn’t happen):
  1. if-clause: Past Perfect
  2. main clause: would/could/may/might have + past participle
  3. main clause for imaginary present result: would + infinitive
I could have been an economist if I had studied economics.
